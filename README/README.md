# Retail & E-commerce Dashboard Project

## Project Overview
This project analyzes retail and e-commerce sales data to identify sales trends, top products, and city-level performance.

## Dataset
Superstore Sales Dataset (51,291 rows)

## Tools Used
- Excel (Data Cleaning)
- SQL (Data Analysis)
- Git & GitHub (Version Control)
- Power BI / Tableau (Dashboard)

## Analysis Tasks
1. Calculate total sales
2. Analyze sales by city
3. Identify best-selling products

## Project Structure
data/ - cleaned dataset  
sql/ - SQL queries  
docs/ - dataset documentation  
images/ - dashboard screenshots
